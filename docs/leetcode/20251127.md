# [3381. 长度可被 K 整除的子数组的最大元素和](https://leetcode.cn/problems/maximum-subarray-sum-with-length-divisible-by-k/)

给你一个整数数组 `nums` 和一个整数 `k` 。

返回 `nums` 中一个 非空子数组 的 **最大** 和，要求该子数组的长度可以 **被** `k` **整除**。

 

**示例 1：**

**输入：** nums = [1,2], k = 1

**输出：** 3

**解释：**

子数组 `[1, 2]` 的和为 3，其长度为 2，可以被 1 整除。

**示例 2：**

**输入：** nums = [-1,-2,-3,-4,-5], k = 4

**输出：** -10

**解释：**

满足题意且和最大的子数组是 `[-1, -2, -3, -4]`，其长度为 4，可以被 4 整除。

**示例 3：**

**输入：** nums = [-5,1,2,-3,4], k = 2

**输出：** 4

**解释：**

满足题意且和最大的子数组是 `[1, 2, -3, 4]`，其长度为 4，可以被 2 整除。

 

**提示：**

- `1 <= k <= nums.length <= 2 * 105`
- `-109 <= nums[i] <= 109`

# 解题思路



```
class Solution {
public:
    long long maxSubarraySum(vector<int>& nums, int k) {
        vector<long long> min_s(k, LLONG_MAX / 2); // 防止下面减法溢出
        min_s.back() = 0;

        long long ans = LLONG_MIN, s = 0;
        for (int j = 0; j < nums.size(); j++) {
            s += nums[j];
            int i = j % k;
            ans = max(ans, s - min_s[i]);
            min_s[i] = min(min_s[i], s);
        }
        return ans;
    }
};
```

复杂度分析


- 时间复杂度：O(n)
- 空间复杂度：O(k)
