2683. 相邻值的按位异或

下标从 0 开始、长度为 n 的数组 derived 是由同样长度为 n 的原始 二进制数组 original 通过计算相邻值的 **按位异或（⊕）**派生而来。

特别地，对于范围 [0, n - 1] 内的每个下标 i ：

  • 如果 i = n - 1 ，那么 derived[i] = original[i] ⊕ original[0]
  • 否则 derived[i] = original[i] ⊕ original[i + 1]
